Hướng dẫn cài Plugin RBZ

Để gỡ hoàn toàn plugin cũ, mọi người tắt SketchUp. Sau đó vào đường dẫn dưới đây và xóa thư mục có tên plugin hoặc file .rb tương ứng.

C:\Users\Tên máy tính của bạn\AppData\Roaming\SketchUp\SketchUp 202x\SketchUp\Plugins

Sau khi xóa xong, mở lại SketchUp và cài file .rbz bản mới trong mục Extension Manager.

1
Tải file .RBZ

Bấm nút tải plugin, lưu file về máy tính.

2
Mở SketchUp

Vào Window → Extension Manager.

3
Install Extension

Chọn file .RBZ vừa tải và bấm cài đặt.

4
Vào menu Toolbar lỗi công cụ ra

Nếu toolbar chưa hiện, tắt mở lại SketchUp.

5
Cập nhật bản mới

Nếu lỗi do bản cũ, gỡ plugin cũ rồi cài bản mới.
